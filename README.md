# Reborn — AI-Powered Life Operating System

Reborn is a personal AI operating system that learns who you are 
and helps you become who you want to be.

Built by Sanchez Bero, 18, Istanbul.

## Features
- Sanchez AI Mentor (chat-based, learns you over time)
- Dynamic Dashboard (modules built through conversation)
- Agent Office (AI automations you can monitor and control)

## Stack
Next.js 14 · TypeScript · Tailwind CSS · OpenAI API · Supabase

## Vision
Every person has untapped potential Reborn exists to unlock it.
